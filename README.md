# OIBSIP Python Programming Internship
## Task 3: Chat Application

### Description
This project is a simple real-time Chat Application developed using Python socket programming.  
It follows a client-server architecture where a server manages multiple client connections and allows users to send and receive messages in real time.

### Technologies Used
- Python
- Socket Programming
- Threading

### Features
- Real-time messaging
- Multiple client connections
- Centralized server communication
- Message broadcasting to all connected users
- Simple terminal-based interface
